# CloudPass Manager

A secure password manager built with Cloudflare Workers, R2, and React.

## Setup
1. Clone the repository: `git clone https://github.com/your-username/cloudpass-manager.git`
2. Install dependencies:
   - Frontend: `cd frontend && npm install`
   - Backend: `cd backend && npm install`
3. Configure `wrangler.toml` files with your Cloudflare credentials.
4. Deploy using GitHub Actions.

## Deployment
- Frontend: Deployed to Cloudflare Pages
- Backend: Deployed to Cloudflare Workers
